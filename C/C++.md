

## 主函数

### 主函数在整个项目中只能出现一次,不区分文件

### 主函数几种标准形式(c语言)

```c++
int main(void)
{
    
    return 0;
}
```

```c++
int main(int argc, char* argv[])
{
    
    return 0;
}
```

### 主函数形式C++

```c++
int main(){
    return 0;
}
```

简写形式：

```c++

main()
{
    
}
编译时自动改成
int main(void)
{
    
}
```
