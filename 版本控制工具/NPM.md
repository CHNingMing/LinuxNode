# NPM

类似Maven，区别管理的是JS代码

## 安装：

NodeJS新版都带NPM工具。

## 模块：

一个模块中包含多个JS文件

### 安装模块：

npm install <模块名> [参数]

默认模块安装到用户目录

### 列出安装模块：

npm list



## package.json

类似Maven的POM.xml，管理各个包的信息

可以通过npm init 创建一个package.json

期间会回答一系列问题：

```
//问题


```

不想回答问题时直接 npm init --yes,直接生成package.jsons
